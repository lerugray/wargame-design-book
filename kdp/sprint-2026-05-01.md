### B.4.2 — Royalty per copy at $24.99: ~$9.80

KDP Amazon-direct paperback formula (60% royalty plan):

```
royalty_per_copy = (list_price × 0.6) - print_cost
```

At launch price $24.99 with estimated print cost ~$5.20 (per
`pricing-recommendation.md`, 362-page B&W 6x9 cream paperback;
KDP's UI shows the authoritative computed value):

- $24.99 × 0.6 = $14.994
- $14.994 - $5.20 = **~$9.80 per copy**

Sanity ladder (estimates; KDP's UI is authoritative):

| List price | Royalty/copy | Notes |
|---|---|---|
| $14.99 | ~$3.80 | Too low for 362-page niche reference |
| $19.99 | ~$6.80 | "Safe discoverability" floor |
| **$24.99** | **~$9.80** | **Recommended launch** |
| $29.99 | ~$12.80 | Defensible post-reviews |

If KDP's pricing page shows **substantially different** than ~$9.80
(e.g. <$8 or >$11), check that the page count hasn't drifted from
362 since the last royalty calc — KDP recomputes print cost from
current page count, and trim/paper-color/page-count drift moves the
royalty.

### B.4.3 — Marketplace currency conversion targets

KDP auto-convert at $24.99 USD will land near these values; KDP
typically rounds to .x99 but lets you override. Targets to manually
adjust toward (when KDP's auto-convert lands at "ugly" values like
£19.97 or €22.49):

| Marketplace | Auto-convert lands ~ | Manual round-to |
|---|---|---|
| Amazon UK (GBP) | £19-£21 (depends on FX) | **£19.99** |
| Amazon DE/FR/IT/ES (EUR) | €22-€24 | **€22.99** or **€23.99** |
| Amazon CA (CAD) | C$33-C$36 | **C$34.99** |
| Amazon AU (AUD) | A$36-A$39 | **A$37.99** or **A$39.99** |
| Amazon JP (JPY) | ¥3,500-¥3,900 | **¥3,799** or **¥3,799** |
| Amazon BR (BRL) | R$120-R$140 | **R$129.90** |
| Amazon MX (MXN) | MX$420-MX$480 | **MX$449** or **MX$499** |
| Amazon IN (INR) | ₹1,800-₹2,100 | **₹1,999** |

The exact KDP auto-convert values will depend on the FX rate at
listing time — sanity-check by glancing at each row in the KDP
pricing page and overriding any that land at memorable-number-misses
per Section B.4 of `launch-checklist.md`. Don't chase precise
currency parity; pricing memory is what matters.

Skip this for marketplaces Ray doesn't expect meaningful traffic
from (probably most non-English ones for a wargame-design title);
KDP's auto-conversion default is fine for those.
